grid-template-rows

Pode utilizar os mesmos valores do grid-template-columns nessa propriedade também.

grid-template-rows: 150px 100px 150px;

Cria 3 linhas no grid, sendo a primeira com 150px, segunda 100px e a terceira 80px. Caso o grid necessite de mais linhas, elas terão o tamanho de acordo com o conteúdo.
