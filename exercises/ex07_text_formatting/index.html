Text Formatting

Bold / Emphasis

In this phrase we have a term in bold using the <b> tag (not semantic).

In this phrase we have a term in emphasis using the <strong> tag (semantic).

Italic / Emphasis

In this phrase we have a term in italic using the <i> tag (not semantic).

In this phrase we have a term in emphasis using the <em> tag (semantic).

Marked Text

You can mark a text using the <mark> tag.

Big Text and Small Text

You can put a big text using the <big> tag (obsolete tag).

You can also put small text using the <small> tag.

Deleted Text and Inserted Text

You can delete a text using the <del> tag.

You can also insert a text using the <ins> tag.

There is also the <u> tag to insert a text (not semantic).

Sup and Sub Text

The <sup> tag is for inserting things like x2+3

And the <sub> tag is to inserting things like H2O
